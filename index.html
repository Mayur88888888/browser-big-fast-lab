browserlab — in-browser LLM engine comparison

Load a model, run the same prompt through four engines side by side — all client-side, weights fetched from Hugging Face, nothing leaves your tab.

How it works. Each lane loads the model independently into the browser's WebGPU device and runs a greedy decode of your prompt, reporting tokens/sec. Models are fetched from Hugging Face on first use (cached by the browser afterward).
